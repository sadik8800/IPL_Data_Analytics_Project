# 🏏 IPL Data Analytics Using Apache Spark

A data analytics project focused on analyzing **Indian Premier League (IPL)** data using **Apache Spark and Databricks**. The project demonstrates how raw IPL datasets can be loaded into Databricks, processed using Spark transformations, and analyzed through data visualization using **Pandas and Matplotlib**.

---

## 📌 Project Overview

The objective of this project is to perform data analytics on IPL data using **Apache Spark** in the **Databricks** environment.

The project starts by uploading IPL datasets into Databricks. Spark is then used to process and transform the data using PySpark. After completing the data transformation and analysis, the processed data is converted into Pandas DataFrames where **Pandas and Matplotlib** are used to create visualizations and explore meaningful insights from the IPL dataset.
